# Media-Stitcher

**Wrapper de FFmpeg para post-producci√≥n de videos de YouTube con soporte GPU**

Media-Stitcher es una utilidad Python que proporciona una interfaz simple y confiable para las operaciones m√°s comunes de manipulaci√≥n multimedia usando FFmpeg, con aceleraci√≥n GPU NVIDIA opcional.

## üéØ Prop√≥sito


### Ejemplos

#### 1. Unir m√∫ltiples videos (intro + cuerpo + outro)

```python

    lista_paths=["intro.mp4", "cuerpo.mp4", "outro.mp4"],

    lista_paths=["video1.mp4", "video2.mp4"],

```

**Par√°metros:**
- `lista_paths`: Lista de archivos a concatenar (en orden)
- `output_path`: Ruta del archivo de salida
- `safe_mode`:
  - `True` (default): concat demuxer - m√°s r√°pido, sin re-encoding, requiere mismo formato/codec
  - `False`: concat filter - m√°s lento, re-encoding, acepta formatos mixtos, **soporta GPU**
- `use_gpu`: Si True, usa aceleraci√≥n GPU NVIDIA (solo con safe_mode=False)
- `show_progress`: Si True, muestra barra de progreso con tqdm

**Par√°metros:**

**Par√°metros:**
- `input_path`: Archivo de entrada
- `start_time`: Tiempo de inicio (string "HH:MM:SS" o n√∫mero en segundos)
- `end_time`: Tiempo de fin (mismo formato que start_time, None para fin del archivo)
- `output_path`: Ruta del archivo de salida
- `stream_copy`: Si True, copia streams (r√°pido). Si False, re-encodifica (preciso)
- `use_gpu`: Si True, usa GPU para re-encoding
- `show_progress`: Si True, muestra barra de progreso

## üéÆ Aceleraci√≥n GPU

Media-Stitcher puede usar GPU NVIDIA para acelerar el procesamiento de video:

### Qu√© se acelera con GPU:

- **Decoding (cuvid)**: Lectura de videos m√°s r√°pida
- **Encoding (nvenc)**: Generaci√≥n de videos m√°s r√°pida
  - **H.264** (h264_nvenc) - Codec est√°ndar
  - **HEVC/H.265** (h265_nvenc) - Mejor compresi√≥n (v0.3.0)
  - **VP9** (vp9_nvenc) - Codec web (v0.3.0, si disponible)
- **Filtros CUDA**: Procesamiento de video acelerado

### Uso de GPU:

```python

    ["video1.mp4", "video2.mp4"],
    "output.mp4",
    safe_mode=False,  # GPU solo funciona con concat filter

)
```

### Detecci√≥n autom√°tica:

Media-Stitcher detecta autom√°ticamente las capacidades GPU de tu sistema:


**Nota**: Si GPU no est√° disponible o falla, la operaci√≥n contin√∫a autom√°ticamente en CPU.

## üêõ Troubleshooting

### Error: "FFmpeg no encontrado en PATH"

Aseg√∫rate de que FFmpeg est√© instalado y accesible:

```bash
ffmpeg -version
```

Si no funciona, agrega FFmpeg a tu PATH o especifica la ruta completa.

### Error: "GPU NVIDIA no disponible"

Verifica que FFmpeg tenga soporte NVIDIA:

```bash
ffmpeg -encoders | grep nvenc
ffmpeg -hwaccels | grep cuda
```

Si no aparecen, necesitas reinstalar FFmpeg con soporte NVIDIA o descargarlo de [gyan.dev](https://www.gyan.dev/ffmpeg/builds/).

### Falla con "safe_mode=True" pero diferentes formatos

Si los videos tienen diferentes codecs/resoluciones, usa `safe_mode=False`:

```python
unir_archivos(["video1.mp4", "video2.avi"], "output.mp4", safe_mode=False)
```

Esto re-encodificar√° los videos (m√°s lento) para compatibilidad.

### ImportError: No module named 'tqdm'

Instala las dependencias:

```bash
pip install tqdm
# o
pip install -e .
```
